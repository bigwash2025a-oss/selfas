# SelfAS 비즈니스 모델 명세서

## 우리 회사의 정체성

**우리는 AS 서비스 제공 업체입니다.**

셀프 세차장 장비(거품기, 세차기, 진공청소기, 에어건 등)가 고장났을 때 출동하여 수리하는 전문 서비스 업체입니다.

---

## 고객 정의

### ❌ 우리 고객이 아닌 사람
- 셀프 세차장을 이용하는 일반 차량 운전자
- 세차장에서 세차하러 온 일반인

### ✅ 우리 고객 (B2B)
- **셀프 세차장 관리자** (사장님)
- **셀프 세차장 대표**
- **세차장 운영 담당자**

---

## 서비스 흐름

```
1. 세차장 장비 고장 발생
   예: 5번 베이 거품기 작동 안됨

2. 세차장 관리자(우리 고객)가 우리 앱으로 AS 요청
   - 베이 번호: 5번
   - 장비: 거품기
   - 문제: 거품이 안나와요
   - 연락처: 010-1234-5678

3. 우리 기사에게 알림 발송
   - 실시간 채팅으로 상황 파악
   - 사진 첨부로 고장 상태 확인

4. 출동 스케줄 조정
   - 기사가 방문 가능 시간 제안
   - 세차장 관리자가 수락/변경/거절

5. 우리 기사 현장 출동
   - 세차장 방문
   - 장비 수리/교체

6. AS 완료 및 히스토리 기록
   - 수리 내용 기록
   - 다음 AS를 위한 데이터 축적
```

---

## 용어 정리

| 용어 | 의미 | 예시 |
|------|------|------|
| **고객** | 세차장 관리자/대표 | 김사장님, 세차장 관리자 |
| **우리 기사** | AS 서비스 제공하는 기사 | 우리 회사 소속 수리 기사 |
| **베이** | 세차장 개별 부스 | 1번 베이, 5번 베이 |
| **장비** | 세차장 설비 | 거품기, 세차기, 진공청소기 |
| **AS 요청** | 세차장 관리자가 보내는 수리 요청 | "5번 베이 거품기 고장" |
| **출동** | 우리 기사가 세차장에 방문 | 현장 방문, 수리 출동 |

---

## 핵심 가치 제안

### 세차장 관리자(고객)에게
1. **빠른 대응**: 장비 고장시 즉시 AS 요청 가능
2. **투명한 소통**: 실시간 채팅으로 진행 상황 확인
3. **유연한 스케줄**: 세차장 영업시간에 맞춰 출동 시간 조정
4. **히스토리 관리**: 장비별 AS 기록 조회 가능

### 우리 회사에게
1. **효율적 배차**: 기사별 일정 관리 최적화
2. **데이터 축적**: 장비별 고장 패턴 분석 가능
3. **품질 관리**: 모든 AS 활동 로그 기록
4. **고객 만족도**: 실시간 소통으로 신뢰 구축

---

## 수익 모델

- **출동 서비스 비용**: 세차장 관리자가 우리에게 지불
- **정기 유지보수 계약**: 월간/연간 계약 옵션
- **긴급 출동 할증**: 야간/휴일 긴급 출동 시 추가 요금

## 시스템 사용자

### 1. 세차장 관리자 (우리의 고객)
- **접속 페이지**: `customer-as-request.html`
- **주요 기능**:
  - AS 요청 접수
  - 우리 기사와 채팅
  - 출동 스케줄 확인/조정
  - 장비 고장 사진 전송

### 2. 우리 기사 (서비스 제공자)
- **접속 페이지**: `technician-app.html`
- **주요 기능**:
  - AS 요청 확인
  - 세차장 관리자와 채팅
  - 출동 스케줄 등록
  - 수리 완료 처리

### 3. 우리 관리자 (내부 운영)
- **접속 페이지**: `staff-dashboard.html`
- **주요 기능**:
  - 전체 AS 현황 모니터링
  - 기사 배차 관리
  - 통계 및 분석
  - 로그 조회

## 기억해야 할 것

**이 시스템의 모든 "고객"은 세차장 관리자를 의미합니다.**
**세차장 이용 고객이 아닙니다.**

우리는 B2B 서비스 업체이며, 세차장 장비 AS 전문 회사입니다.
